Calorie Detection and Guide to Food
Abstract
This project aims to provide insights into food and beverages for those who are health conscious. We used technologies like Machine Learning and Image Processing along with Android as a platform for making an application. The application takes a picture from the user and calculates calories from the picture of food and shows to the user using a neural network algorithm. Apart from that, it provides nutritional information about food so that the user can maintain dietary information.
Technology used
  • Android
  • Machine Learning
  • OpenCV
  • CNN
