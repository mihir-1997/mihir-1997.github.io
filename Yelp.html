Yelp Prototype
Abstract

Yelp Prototype is a clone of the real Yelp app that concentrates on Restaurant-Customer relation where a restaurant lists dishes and user orders from it. The restaurants can register, login, update profile, add/update dishes, and add future events. The users can register, login, update profile, search for restaurants, order from restaurants, review restaurants, and register for events. The application has functionalities like search/filter/sort restaurants based on delivery methods and ratings, orders filtering for both restaurants and users, and sort events by date. To enhance user experience of the app, client-side pagination is used for listing of restaurant dishes and events. The app also has Google Maps embadded in it so that just by a click, a user would get the direction. One of the highlighting features of the app is its real-time chat system which is developed using socket programming where only a restaurant can initiate the chat and user can reply back.
