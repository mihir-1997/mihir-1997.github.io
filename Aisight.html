AISight
Abstract
AISight is a project aimed to aid visually impaired people to see their surroundings better. It is a small RaspberryPI based device with a camera module and a power bank attached to it making it extremely portable, which describes the surroundings in the human voice. The device continuously takes images of surrounding and sends it to a server and the server sends a description of the image back to the device which is converted into human voice and output is given through either headphone or speaker. The model which is a Convolutional Neural Network followed by Recurrent Neural Network runs on the cloud which converts an image to human-readable sentences. In other words, rather than detecting objects from an image which is the past trend, the model generates the caption of the image. As preprocessing part, we compress the image so even if the internet connectivity is slow, it can generate audio quickly. For more details visit Github.
